## About

I build things that matter.

A distributed hospital system with production-grade **HL7 v2 / FHIR R4** interoperability. An RL training environment where a GPT-2 policy learns — through reward signals and regression backtracking — to write better code than it was given. A cycle tracking application that encrypts **100% of its records before they ever touch a database**.

I'm a **B.Tech CSE (AI-ML)** student at Adani University (9.20 CGPA), simultaneously completing **IIT Madras's Diploma in Programming**. I write technical blogs, reach hackathon finals, and contribute to open source — not to pad a résumé, but because the problems are genuinely interesting.

> *I literally cannot sleep if there is unresolved code on my desk.*
